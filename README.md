# Smart File Organizer

A Java application that automatically organizes files into folders based on their extensions.

## Features

- Organize files by extension
- Automatic folder creation
- Conflict name resolution
- Logging of moved files
- Undo last operation
- JavaFX graphical interface
- Error handling

## Technologies

- Java 21+
- JavaFX
- Maven
- Jackson
- Logback

## How to Run

### Clone repository

```bash
git clone https://github.com/yourusername/SmartFileOrganizer.git
```

### Build

```bash
mvn clean install
```

### Run

```bash
mvn javafx:run
```
